
#  News Website

News website using Newsapi



## 📍 Run Locally

Clone the project

```bash
  git clone https://github.com/Tushar-goyal-2712/News-website.git
```

Install node_module dependencies

```bash
  npm install
```

Start the server
```bash
  npm run start
```

Visit [Local Host](http://localhost/)
